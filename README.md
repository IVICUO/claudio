# Claudio

Claudio, IVICUO's AI brain.

A three-layer system for IVICUERs: a React chat app on GitHub Pages, a Cloudflare Worker proxying Anthropic and tool APIs, and a Claude Code environment with shared MCP connections and scheduled GitHub Actions agents.

See `docs/architecture.md` for the full picture, `docs/onboarding.md` to get started in under five minutes.
